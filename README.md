# homebrew-tap

Homebrew formulae for richclement tools.

## Installation

```bash
brew tap richclement/tap
brew install ralph-cli
```

## Install Packages

```bash
# formula
brew install richclement/tap/<name>
```

## Available Formulae

- `ralph-cli` - CLI tool for ralph

## Update / Uninstall

```bash
brew update
brew upgrade

brew uninstall <formula>
```
